1️⃣ 프로그램 소개

이 프로그램이 하는 일

이 프로그램은 쇼핑몰에서 팔 물건의 사진을 자동으로 예쁘게 만들어줍니다.

마치 전문 사진작가처럼:

  • 📷 배경을 깨끗하게 제거
  • 🌑 그림자를 자연스럽게 추가
  • ✨ 밝기와 색상을 조정
  • 🎯 물건을 정확한 중앙에 배치
  • 💾 최종 이미지를 저장
이 프로그램은 AI(인공지능)를 사용해서 작업합니다. 마치 똑똑한 비서처럼 사진을 분석하고 개선해줍니다!

2️⃣ 시작하기

프로그램 실행하기

1 Windows 시작 메뉴에서 "명령 프롬프트" 또는 "PowerShell" 열기
2 cd D:/CLAUDE_CODE_WORK/shop-image-editor 입력 후 Enter
3 python gui.py 입력 후 Enter
4 조금 기다리면 프로그램 창이 열립니다 ✨

3️⃣ 메인 화면 설명

화면 구성

프로그램을 실행하면 여러 개의 탭(버튼)이 보입니다:

탭 이름 무엇을 하나요?
실행 사진을 선택하고 처리를 시작하는 곳
프롬프트 편집 AI에게 주는 명령을 바꾸는 곳
그림자 힌트 각 물건 종류별로 어떻게 그림자를 그을지 알려주는 곳
설정 API 키나 결과물 크기 같은 옵션을 바꾸는 곳

📸 다양한 디자인 스타일

프로그램은 3가지 디자인 스타일을 지원합니다:

1️⃣ 모던 라이트 스타일 (사용하기 쉬움)

파란 사이드바와 깔끔한 카드 레이아웃입니다.

2️⃣ 다크 글래스모피즘 스타일 (고급스러움)

어두운 배경에 보라색 포인트로 럭셔리한 느낌입니다.

3️⃣ 미니멀 클린 스타일 (Apple 스타일)

맨 위의 네비게이션 바와 여백이 많은 깔끔한 디자인입니다.

4️⃣ 기본 사용법

Step 1: 이미지 파일 선택하기

1 "실행" 탭에서 입력 부분의 "폴더 선택" 버튼 클릭
2 편집할 사진이 있는 폴더를 선택
3 "확인" 버튼 클릭

Step 2: 결과물이 저장될 위치 정하기

1 출력 부분의 "폴더 선택" 버튼 클릭
2 완성된 사진을 저장할 위치 선택
3 "확인" 버튼 클릭

Step 3: 처리 시작하기

1 필요한 설정을 한 후 ▶ 실행 버튼 클릭
2 프로그램이 사진을 처리합니다 (시간이 좀 걸릴 수 있습니다)
3 완료되면 결과물이 출력 폴더에 저장됩니다 ✅
여러 개의 사진을 한 번에 처리하려면 입력 폴더에 모든 사진을 넣으면 됩니다!

5️⃣ 이미지 검증 상세 가이드

"이미지 검증"이란 AI가 처리한 사진이 잘 만들어졌는지 확인하고 필요하면 다시 처리하는 것입니다.

🔍 검증이란?

프로그램이 다음을 자동으로 확인합니다:

  • ✅ 배경이 제대로 제거되었나?
  • ✅ 그림자가 자연스럽나?
  • ✅ 물건이 정중앙에 있나?
  • ✅ 밝기가 적당한가?

📊 점수 이해하기

점수 의미 어떻게 하나요?
90점 이상 🟢 완벽합니다! 그냥 사용해도 됩니다
70~90점 🟡 괜찮지만 개선 가능 자동 수정을 시도하거나 수동으로 조정
70점 미만 🔴 문제가 있어요 반드시 다시 처리해야 합니다

🔧 자동 수정하기

점수가 낮으면 프로그램이 자동으로 다시 처리할 수 있습니다. 마치 의사의 2차 진료처럼, AI들이 함께 모여 어떻게 개선할지 의논합니다.

1 점수가 낮은 사진을 선택
2 자동 수정 버튼 클릭
3 "6단계 회의" 윈도우가 열립니다
4 AI들이 대화하는 것을 지켜봅니다 (2~3분 소요)
5 완료되면 새로운 점수가 표시됩니다
6단계 회의란?
• 1단계: 각 AI가 의견 제시
• 2단계: 서로 의견 검토
• 3단계: 공통 문제 파악
• 4단계: 해결방법 제시
• 5단계: 방법 토론
• 6단계: 최종 결정
마치 회의실에서 여러 사람이 의논하는 것처럼!

💬 회의 중에 의견 주기

회의 창의 맨 아래에 입력창이 있습니다. 여기에 메시지를 입력하면 AI들이 당신의 의견을 반영합니다!

1 입력창에 당신의 생각을 입력 (예: "배경을 더 밝게 해주세요")
2 Enter 키 또는 "전송" 버튼 클릭
3 AI들이 당신의 의견을 읽고 반영합니다
구체적으로 말할수록 좋습니다!
"더 밝게" 보다는 "명도를 50 더 올려주세요"처럼 말하면 더 정확해요!

6️⃣ 프롬프트 편집 완벽 가이드

🎯 프롬프트란?

"프롬프트"는 AI에게 주는 명령입니다. 마치 요리사에게 "맛있는 국을 만들어주세요"라고 말하듯이, 프로그램에서 "이 사진을 이렇게 처리해주세요"라고 AI에게 말하는 것입니다.

📝 프롬프트 종류

프롬프트 언제 사용하나요?
시스템 프롬프트 AI의 기본 역할을 정해요 (예: "전문 사진작가처럼")
분석 요청 사진을 어떻게 분석할지 말해줘요
검증 프롬프트 처리된 사진이 좋은지 평가하는 방법을 알려줘요

✏️ 프롬프트 편집하는 방법

1 "프롬프트 편집" 탭 클릭
2 수정하고 싶은 프롬프트 찾기
3 텍스트 박스에서 원하는 부분 수정
4 저장 버튼 클릭
5 기본값 복원을 누르면 원래대로 돌아갑니다

📚 좋은 프롬프트 작성 팁

구체적으로 써야 해요!
❌ "예쁘게 해줘"
✅ "명도를 높이고, 배경을 완전히 제거하고, 물건을 중앙에 배치해줘"
단순명료하게 써야 해요!
❌ "음... 이 사진이 좀 어둡고 배경이 지저분한데 어떻게 할까요?"
✅ "배경을 제거하고 밝기를 증가시켜주세요"
명확한 동작을 명령해야 해요!
❌ "좀 더..."
✅ "명도를 30 증가, 채도를 20 증가, 대비를 15 증가"

2. 신발 아래에 자연스러운 그림자 추가

1. 배경을 깨끗하게 제거
2. 가방의 질감이 잘 보이도록 조명 조정
3. 가방을 살짝 45도 각도로 배치

🔍 프롬프트 테스트하기

프롬프트를 수정한 후에는 한두 개의 사진으로 테스트해보는 것이 좋습니다.

1 새 프롬프트로 한 장의 사진 처리
2 결과가 좋으면 여러 장을 처리
3 결과가 나쁘면 프롬프트를 수정하고 다시 시도
기본값 복원을 누르면 모든 편집이 사라집니다. 주의해서 사용하세요!

7️⃣ 설정

API 키 관리

API 키는 AI 서비스를 사용하기 위한 비밀번호같은 것입니다.

1 "설정" 탭 클릭
2 각 AI 서비스의 키 입력 (Claude, ChatGPT, Gemini 등)

API 키는 절대로 다른 사람에게 알려주면 안 됩니다! 마치 은행 비밀번호처럼 안전하게 관리하세요.

8️⃣ 자주 묻는 질문 (FAQ)

Q1: 프로그램이 느린데요?

A: AI가 열심히 일하고 있는 거예요. 한 장당 1~5분 정도 걸리는 것이 정상입니다. GPU가 있으면 더 빨라집니다!

Q2: 배경 제거가 잘 안 돼요

A: 다음을 시도해보세요:
• 프롬프트에서 배경 제거 관련 부분을 더 자세하게 작성
• 자동 수정 기능 사용
• 다른 배경 제거 서비스 시도 (Photoroom → remove.bg)

Q3: 그림자가 부자연스러워요

A: 그림자 힌트를 추가해보세요:
• "그림자 힌트" 탭에서 물건 종류별로 그림자 가이드 작성
• 프롬프트에서 그림자 처리 방식 구체적으로 명시

Q4: 결과가 마음에 안 들어요

A: 자동 수정을 시도하세요!
• 사진 선택 후 "자동 수정" 클릭
• 6단계 회의 중에 당신의 의견 입력
• AI들이 피드백을 반영해서 개선합니다

Q5: API 키를 어디서 얻나요?

A: 각 AI 서비스의 웹사이트에서:
• Claude: anthropic.com
• ChatGPT: openai.com
• Gemini: google.com/ai

Q6: 한 번에 몇 개의 사진을 처리할 수 있나요?

A: 입력 폴더에 있는 모든 사진을 한 번에 처리할 수 있습니다! 자동으로 하나씩 처리합니다.

더 많은 도움이 필요하면 프로그램 폴더의 "history.md" 파일을 읽어보세요!
